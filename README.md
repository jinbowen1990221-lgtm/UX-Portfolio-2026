# UX Portfolio 2026

一个以复古游戏机和卡带交互为核心概念的个人作品集网站，用游戏世界的叙事方式展示 UX / 产品设计经历、项目案例与联系方式。网站结合 3D Game Boy 模型、可交互卡带、像素风视觉语言和响应式页面布局，呈现一个更具记忆点的设计师个人品牌入口。

在线访问：<https://jinbowen1990221-lgtm.github.io/UX-Portfolio-2026/>

## Project Description

A game-inspired UX/Product Design portfolio for Bowen Jin, built around a retro console and cartridge interaction. It presents professional experience, selected UX case studies, creative coding work, and contact information through an interactive 3D and pixel-art web experience.

## Highlights

- Retro game world concept with 3D Game Boy and cartridge interactions
- UX/Product Design portfolio structure for resume, case studies, and creative coding work
- Pixel-art visual direction with responsive layout and animated interface details
- Public GitHub Pages deployment for easy sharing
